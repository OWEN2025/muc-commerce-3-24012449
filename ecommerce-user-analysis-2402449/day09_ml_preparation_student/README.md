## 用自己的话回答

- 什么是特征，什么是标签：特征是模型在判断时能看到的输入信息；标签是我们希望模型预测的结果，比如这里的 `Churn`。
- 为什么要保留测试集：测试集要留到最后再用，这样才能模拟模型面对“新用户”时的真实表现，避免把答案提前泄露给模型。
- 为什么83%准确率仍可能没有用：因为如果数据里“未流失”本来就占多数，模型只要一直猜“未流失”也可能拿到较高准确率，但它根本抓不到真正会流失的用户。
